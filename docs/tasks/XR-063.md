# XR-063: стенд сравнения с Xray/Amnezia

Чтобы «на голову лучше» было измеримым, а не декларацией.

## Что сделать

Воспроизводимые прогоны: throughput на чистом канале и на lossy (эмуляция
`tc netem`: потери/задержка/джиттер), latency установки соединения (cold start,
было до 20с на Android), CPU и память на ARM-роутере (musl), поведение под
потерями против одиночного TCP. Плюс детект-тесты: энтропия первого пакета,
реакция на активное зондирование, сравнение с эталонными Xray (VLESS+Reality) и
AmneziaWG на том же стенде.

Прогонять до и после каждого шага v2 (XR-061/062/064), ловить регресс. Рычаг:
валидирует всю программу v2.

## DoD

Скрипты стенда + базовый отчёт с числами по нам и по двум конкурентам.

Смежно с XR-060.
