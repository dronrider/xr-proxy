# XR-055: кривая компоновка карточки шары в «Моих шарах»

Название втискивается в узкую колонку и обрезается («Антон Назаров - О...»),
хотя карточка уже трёхстрочная и место под имя есть.

## Причина (регрессия XR-043)

Один Row с ведущей иконкой + Column(weight 1f) + трейлинг-кластер из трёх
контролов (иконка «папка хранения» + тумблер синка + корзина); кластер ~150dp
плюс ведущая иконка ужимают колонку имени до ~110dp.

## Что сделать

Дать имени полную ширину без роста высоты блока: либо вынести заголовок в
отдельную верхнюю строку на всю ширину, а метаданные (выбрано/папка) + контролы
под ней; либо убрать третью иконку из кластера, сделав строку «Папка: ...»
тапабельной для смены папки (заголовку больше места).

Смежно с XR-044 (та же карточка/экран).
