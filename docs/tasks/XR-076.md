# XR-076: мессенджер как сервис экосистемы (болванка LLD на будущее)

Чат поверх федерации хабов (не проприетарный глобальный сервер, а связь хабов,
класс Matrix), E2E-крипта групп (ориентир MLS), ориентир по фичам Signal.

## Ключевые отличия

- быстрый и безопасный перенос истории на новые устройства (у Signal боль);
- продвинутый быстрый поиск + срезы «со ссылками/медиа» (телеграм-подобно);
- кворум для управления группами;
- глубокая кастомизация вида, маскировка иконки под другое приложение;
- эфемерность по политике хаба/чата;
- продвинутое избранное, эффективные треды.

## Решено с владельцем

Федерация гибрид (MLS + свой транспорт), кворум это управление группой
(демократическая модерация), камера-детекция отложена в research. Осталось
обсудить: ключ бэкапа, глубину маскировки, подачу удаления, объём избранного.

## Приоритизация

Далёкий сервис, зависит от XR-058 (навигация), XR-030/074 (идентичность), крипта
смежна с XR-061. Серьёзность 0, Ценность высокая (флагманский сервис),
Неопределённость максимальная.
