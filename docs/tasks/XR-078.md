# XR-078: каналы обновления (stable/beta/alpha) для APK

Собранный apk уходит на хаб без вопросов, но автообновление получают только
подписанные на канал, а не все пользователи.

## Спроектировать

- хаб хранит релизы и подписанный манифест ПО КАНАЛАМ
  (`/api/v1/app/latest?channel=` или отдельные пути);
- `xr-hub sign-release --channel`;
- приложение хранит выбранный канал (дефолт stable) и проверяет latest своего
  канала, переключатель канала в настройках.

Обычные пользователи на stable не видят beta/alpha сборки. Смежно с XR-041
(индикация обновления), XR-058 (настройки под шестерёнкой).
