# XR-052: импорт контента по URL как плагин (поверх записи XR-051)

## UI

В writable-шаре зайти в папку -> кнопка «Импорт по URL» -> вставить ссылку на
страницу -> агент качает контент в эту папку с адекватным именем из метаданных
(напр. заголовок ролика).

## Архитектура

НЕ прибито к xr: агент держит общий интерфейс плагина-фетчера (вход
{url, dest_dir}, выход прогресс + итоговый файл), референс-плагин это обёртка
yt-dlp+ffmpeg, ядро xr-share остаётся тонким файлсервером без жёсткой
зависимости; роутинг URL -> плагин по паттерну, плагины опциональны (владелец
включает).

## Безопасность

Агент в чужой квартире, URL от пользователя: dest строго в пределах шары, url
литеральным аргументом без shell, SSRF-защита, лимиты размер/диск/таймаут,
только write-скоуп, по возможности песочница плагина.
