# XR-041: индикация новой версии из любого места приложения

Уведомление о новой версии в рамках всего приложения, а не только на главной
(VPN).

## Проблема

Сейчас баннер обновления виден лишь на главном экране. Плюс сама авто-проверка
ненадёжна: баннер часто не приходит, приходится проверять вручную (фидбек,
подтверждено).

## Что сделать

- Индикация доступна с любой вкладки: глобальное состояние «доступно
  обновление» + бадж/точка на иконке нижней навигации.
- Переработать отображение, один из вариантов это свести настройки в одно место
  и показывать бадж на иконке настроек (дизайн открыт). Решить, на какой
  иконке/в каком разделе показывать.
- Сделать проактивную проверку надёжной.
