# XR-062: стойкость к DPI-фингерпринту первого пакета

Класс AmneziaWG.

## Что сделать

Убрать детерминированный magic ([protocol.rs](../../xr-proto/src/protocol.rs),
`HEADER_MAGIC 0xA0`), рандомизировать размеры и тайминг хендшейка, добавить
конфигурируемые junk-пакеты и pre-handshake padding, чтобы поток не опознавался
энтропийными эвристиками и по первому пакету (popcount, доля печатаемых,
случайность первых N байт). Параметры пер-деплой (аналог Jc/Jmin/Jmax/S1/S2 у
AmneziaWG). Ложится поверх криптоядра XR-061.

## Приоритизация

Серьёзность 0 пока обход работает, Ценность высокая (ось не-заблокироваться,
владелец ценит её выше приватности). Дешёвая полумера против ТСПУ, потолок
неотличимости даёт TLS-камуфляж XR-057. Зависит от XR-061, смежно с XR-057.
