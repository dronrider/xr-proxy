# XR-095. Пауза в авиарежиме врёт: «доверенная сеть» и «в сети ограничения» при полном отсутствии связи

## Симптом

Телефон стоит в авто-паузе на доверенной сети. Включаем авиарежим: главная
продолжает показывать «На паузе · Доверенная сеть · трафик идёт напрямую», а
ниже загорается оранжевое «В этой сети есть ограничения: включите VPN, если
что-то не открывается». Сети при этом нет вообще, ни ограничений, ни
«напрямую» (фидбек владельца 2026-07-10, заметил при проверке XR-093).

## Корень

Фаза Paused переживает исчезновение сети (это правильно: резюмить VPN в
никуда незачем, а `maybeEvaluate` реагирует только на появление/смену
дефолтной сети). Но подписи под статусом остаются от последней доверенной
сети, а перезапущенная проба ограничений (`RestrictionProbe`, ре-пробы по
экрану/foreground/таймеру) на мёртвом аплинке фейлит все хосты и выставляет
`restrictedNetwork`: для неё «всё недоступно из-за DPI» и «всё недоступно,
потому что сети нет» неразличимы.

## Что сделать

- Различать «дефолтной сети нет» (авиарежим, пропавший Wi-Fi) и «сеть есть,
  но с ограничениями». Сигнал: `ConnectivityManager.activeNetwork == null`
  (или onLost без последующего onAvailable) перед запуском пробы и в момент
  показа подписи.
- Без сети пробу не запускать (результат заведомо мусорный), флаг
  `restrictedNetwork` сбрасывать.
- В Paused без сети заменить блок подписи на честное состояние в духе
  «Сети нет, VPN включится сам, когда связь вернётся» (без «доверенной сети»
  и без оранжевой строки). Точные тексты согласовать с владельцем перед
  реализацией (AskUserQuestion, урок XR-049).
- Проверить смежный кейс: Connected + авиарежим (что показывает главная,
  пока движок ждёт сеть) - вероятно, отдельная строка того же состояния.

## Сценарий проверки

1. Авто-пауза на доверенной сети (статус «На паузе», подпись про доверенную
   сеть).
2. Включить авиарежим. Ожидаемо: подпись меняется на «сети нет» без
   упоминания доверенной сети, оранжевого предупреждения об ограничениях не
   появляется ни сразу, ни после ре-проб (подождать пару минут, экран
   выключить/включить).
3. Выключить авиарежим, вернуться на ту же доверенную сеть. Ожидаемо: пауза
   с прежней подписью «Доверенная сеть», проба отрабатывает штатно.
4. Уйти с доверенной сети на LTE. Ожидаемо: VPN поднимается сам (как раньше).
